## What Happens When You Use This Phrase

1. **Manus scans** the entire `/home/ubuntu/` sandbox
2. **Creates a dated backup folder** in `manus-task-backups/backups/`
3. **Copies all files** preserving directory structure
4. **Generates CONTINUATION.md** with full context
5. **Creates manifest.json** with file inventory
6. **Pushes to GitHub** for permanent storage
7. **Provides the continuation prompt** ready to paste into a new session
